# B-ASM-Language

## 変数宣言

```txt
let 変数名=初期値;
````

複数宣言:

```txt
let a=1,b=2,c=3;
```

---

## 代入

```txt
変数名=式;
```

例:

```txt
a=a+1;
x=10*5;
```

---

## 文字列出力

```txt
put "文字列";
```

例:

```txt
put "hello";
```

---

## 変数出力

```txt
put 変数名;
```

例:

```txt
put a;
```

---

## 入力

```txt
inp 変数名;
```

例:

```txt
inp x;
```

---

## ラベル定義

```txt
ラベル名:
```

例:

```txt
loop:
```

---

## 無条件ジャンプ

```txt
jmp ラベル名;
```

例:

```txt
jmp loop;
```

---

## 条件ジャンプ

```txt
if(条件)ラベル名;
```

使用可能比較演算子:

* `==`
* `!=`
* `>`
* `<`
* `>=`
* `<=`

例:

```txt
if(a<10)loop;
```

---

# 演算子

* `+`
* `-`
* `*`
* `/`

例:

```txt
a=1+2;
b=a*10;
c=b/2;
```

---

# コメント

```txt
// コメント
```

例:

```txt
put "hello"; // コメント
```
